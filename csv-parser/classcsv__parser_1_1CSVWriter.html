Detailed Description

Class for writing CSV files.

See csv_parser::csv_escape() for a function that formats a non-CSV string.

To write to a CSV file, one should

  1. Initialize a CSVWriter with respect to some file
  2. Call write_row() on std::vector<std::string>s of unformatted text
  3. close() the file handle
